## Iris Data Analysis Project

A Python data analysis project that loads a dataset directly from a URL and performs exploratory data analysis (EDA) and visualization.

## Project Overview
## This project demonstrates:

- Loading dataset from an online URL
- Exploring dataset structure
- Performing summary statistics
- Counting categorical values
- Creating a basic scatter plot visualization
- The dataset used is the famous Iris Flower Dataset.

## Dataset Source:
https://raw.githubusercontent.com/mwaskom/seaborn-data/master/iris.csv

## Technologies Used

- Python 3
- Pandas
- Matplotlib

## Project Structure

- iris_analysis_project/
- iris_analysis.py
- README.md

## Installation
- Make sure Python is installed.
- Check version:
- python --version

## Install required libraries:
- python -m pip install pandas matplotlib

## How to Run

- Download or clone the project.
- Open terminal inside the project folder.

## Run:
- python iris_analysis.py

## What This Script Does
- Loads Iris dataset from URL
- Displays first 5 rows
- Shows dataset information
- Displays summary statistics
- Counts number of species
- Generates scatter plot (Sepal Length vs Petal Length)
